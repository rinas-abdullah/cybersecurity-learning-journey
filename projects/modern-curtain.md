# 🚀 Modern Curtain – Smart Curtain Ordering System

## 📌 Overview
Modern Curtain is a smart digital system designed to automate customer interactions and ordering processes through a professional web platform integrated with a WhatsApp bot.

The system reduces manual workload, improves response time, and enhances customer experience.

---

## 🎯 Problem Statement
Modern Curtain receives a high volume of customer requests via WhatsApp, leading to:

- Lost or untracked requests  
- Delayed responses  
- Difficulty managing customer data  

---

## 💡 Solution
A fully integrated automation system was developed, consisting of:

| Component       | Description |
|----------------|------------|
| 🌐 Website      | Displays services, company branches, Google Maps, and contact information |
| 🤖 WhatsApp Bot | Automatically collects customer requests and responds instantly |
| 🗄 Database     | Stores and manages customer data securely |

---

## 🌍 Website Features
- Modern, responsive UI design  
- Service showcase (Curtains, Smart Curtains, Custom Designs)  
- Integrated Google Maps for branch locations  
- Social media integration (YouTube, TikTok, Instagram, WhatsApp)  

---

## 🤖 WhatsApp Bot Features
The bot automatically collects customer information:

| Field |
|------|
| Customer Name |
| Curtain Type |
| Location |
| Activity Type (Residential / Commercial) |

All collected data is stored directly in the database for processing.

---

## ⚙️ Technologies Used
- HTML, CSS, JavaScript

- Backend (Node.js / Python – optional)  
- Database (MySQL / Firebase)
